# Fashion-landing

### 📋 Prerequisites

- Node.js v18+
- npm v9+ or yarn v1.22+
- Git

### 🛠️ Technology Stack

- Vue.js 3 (Composition API)
- Vite
- GSAP

### ⚙️ Installation

```bash
git clone https://github.com/your-username/fashion-site.git
cd fashion-site
npm install
```

### 💻 Development

```bash
npm run dev
```

### 🏗️ Production Build

```bash
npm run build
npm run preview
```
